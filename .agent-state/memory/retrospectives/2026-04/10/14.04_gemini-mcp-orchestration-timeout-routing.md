Before writing this note, the local pulse snapshot showed 4 total sessions, 1 recorded session today, 3 active days, and a 2-day streak on `main`, with a +100.0% weekly session change. ## Lessons Learned
- Fixing orchestration routing can be safer than extending timeouts inside a nested transport stack.
- When a failure pattern matches an SDK default almost exactly, treat architecture as suspect before treating runtime budgets as suspect.
- Parallel research pays off, but only if I collapse the findings into one decisive model quickly.
