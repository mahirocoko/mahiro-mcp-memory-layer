## Lessons Learned

- `oh-my-openagent` demonstrates a practical Claude hooks compatibility subset for OpenCode, centered on `PreToolUse`, `PostToolUse`, `UserPromptSubmit`, `Stop`, and `PreCompact`.
- The useful abstraction is an adapter layer: keep OpenCode as runtime, read Claude-style settings, emit Claude-shaped hook input, and translate hook outputs back to OpenCode behavior.
- This memory-layer repo should keep hook-runtime knowledge as research context only; it should not promise hook execution or workflow orchestration.
- Broad file searches should avoid `.agent-state` symlink-heavy paths when possible.
