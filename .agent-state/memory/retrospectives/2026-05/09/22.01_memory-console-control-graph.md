## Lessons Learned

- For destructive local-console features, pair service-level data integrity tests with route-level integration tests against temporary stores.
- Derived graph UI should stay visibly read-only: GET filters only, grouped projection output, no relationship editing, and no persistence path.
- Product identity migrations need two grep layers: current product surface and historical local memory/retrospective artifacts.
- When final-wave gates approve but plan text requires a human okay, document the instruction conflict before marking completion if a higher-priority continuation directive overrides it.
